    SSAI : Détection de blocage publicitaire et basculement automatique

    Dans cette rubrique, vous apprendrez comment activer le paramètre de basculement automatique d'un lecteur Brightcove pour demander des annonces SSai (Server Side Ads Insertion) lorsque des bloqueurs d'annonces sont détectés.

    Présentation

    Lorsque le basculement publicitaire est activé, lorsque Brightcove Player détecte un bloqueur de publicité présent dans le navigateur, le lecteur demandera automatiquement un flux SSAI. Sinon, il utilisera les annonces IMA.

    Le basculement publicitaire SSAI requiert les éléments suivants :

    • Plugin IMA3 v3.2.0+
    • Plugin SSAI v1.3.1+
    • Brightcove Player version 6.8.0+

    Avertissements

    • La détection des bloqueurs d'annonces est très fiable, mais pas 100%.
    • Les clients qui utilisent cette fonctionnalité doivent utiliser la livraison dynamique.
    • Seuls les plugins IMA et SSAI sont pris en charge (pas de FreeWheel).

    Premiers pas

    Pour lire des annonces côté serveur à partir de Video Cloud, procédez comme suit :

    1. Créer une configuration d'annonce
    2. Créer un lecteur Brightcove
    3. Mise en œuvre du basculement automatique à l'aide

    Créer une configuration d'annonce

    La configuration de l'annonce définit divers aspects de la lecture SSAI, y compris les appels publicitaires, les balises et d'autres options de configuration. La réponse de votre annonce peut être des règles de publicité VAST, VMAP ou DFP. Pour créer une configuration d'annonce, procédez comme suit :

    1. Dans Video Cloud Studio, développez le menu ADMIN et sélectionnez Paramètres publicitaires côté serveur.

    2. Ajoutez des informations pour la configuration de votre annonce et sélectionnez Enregistrer.

    Pour plus d'informations, consultez le document Configuration des paramètres publicitaires côté serveur .

    Pour créer une configuration d'annonce à l'aide de l'API SSAI, consultez le document API Video Cloud SSAI Ad Config .

    Créer un lecteur Brightcove

    Créez un nouveau lecteur Brightcove à l'aide de Video Cloud Studio. Pour plus de détails, reportez-vous à la section Démarrage rapide : Création et style d'un document Player .

    Vous devez utiliser une version 6.8.0+ de Brightcove Player.

    Mise en œuvre du basculement automatique à l'aide

    Le moyen le plus simple de configurer votre lecteur pour les annonces de basculement automatique est avec Video Cloud Studio. Une fois que vous avez créé une configuration publicitaire et un lecteur, vous êtes prêt à configurer le lecteur pour le basculement automatique comme suit :

    1. Ouvrez le module PLAYERS et localisez le lecteur auquel vous souhaitez ajouter des fonctionnalités publicitaires.
    2. Sélectionnez le lien pour que le joueur ouvre ses propriétés.
    3. Sélectionnez Publicité dans le menu de navigation de gauche.
    4. Cochez la case Activer côté client (IMA) .

    5. Incluez l'URL de votre balise publicitaire IMA. Pour cet exemple, nous utiliserons l'exemple d'URL de balise d'annonce.

      Pour plus d'informations sur les propriétés publicitaires du lecteur, consultez le document Configuration de la publicité du lecteur à l'aide du module Players .

    6. Cochez la case Activer le côté serveur (SSAI) .

      L'option côté serveur n'apparaîtra pas à moins que votre compte n'ait été activé pour SSAI.
    7. Dans le menu déroulant Sélectionner une configuration , sélectionnez la configuration d'annonce que vous souhaitez associer à ce lecteur.
    8. Si vous souhaitez que les superpositions s'affichent sur vos annonces, cochez la case Activer les superpositions d'informations publicitaires . Cela inclut les superpositions « En savoir plus » et le décompte des annonces.

    9. La détection des blocs publicitaires avec basculement d'annonces côté serveur est automatiquement activée lorsque vous activez les annonces côté client et côté serveur.

    10. Sélectionnez Enregistrer.

    11. Pour publier le lecteur, sélectionnez Publier et intégrer > Publier les modifications.

      C'est tout ce que vous devez faire pour activer le basculement automatique pour votre joueur.

    Lire une vidéo avec des annonces

    Toute vidéo que vous récupérez à partir de Video Cloud qui a été ingérée avec la diffusion dynamique inclut les annonces IMA côté client lorsqu'aucun bloqueur d'annonces n'est détecté. Lorsqu'un bloqueur d'annonces est détecté, votre vidéo inclut les annonces côté serveur spécifiées dans le fichier VMAP dans la configuration de votre annonce. Notez que la vidéo doit être associée à une piste audio pour que SSAI fonctionne.
