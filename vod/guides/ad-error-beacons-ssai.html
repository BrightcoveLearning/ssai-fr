Vue d'ensemble

Avec SSAI, les annonces sont intégrées à votre flux de contenu côté serveur. Le balises des événements d'erreur d'annonce VAST est fourni lors de l'utilisation de VOD + SSAI.

Les événements d'erreur publicitaires fournissent des informations sur vos workflows publicitaires, ce qui vous permet de résoudre les problèmes de manière proactive. Veiller à ce que vos annonces soient diffusées contribuera à maximiser les revenus publicitaires.

Balises d'erreur d'annonce

Le tableau suivant définit les événements d'erreur publicitaires balises via SSAI, tels que définis dans les spécifications VAST 3.0/4.0.

Code d'erreur Type Description
100 ErrorCodeXMLParse

Erreur d'analyse XML

  • Erreur sur la transformation XSLT
  • La réponse était vide
200 ErrorCodeTrafficking

Erreur de trafic. Le lecteur vidéo a reçu un type d'annonce qu'il ne s'attendait pas et/ou ne peut pas afficher.

  • Réponse d'annonce valide renvoyée, mais aucune annonce linéaire n'a été trouvée
  • Un format d'annonce non progressif a été renvoyé
    • VPAID swf (lorsqu'il n'est pas prévu en tant que pass-through)
    • Type Mime vidéo non valide
301 ErrorCodeWrapperRequest

Délai d'expiration de l'URI VAST fourni dans l'élément Wrapper, ou de VAST URI fourni dans un élément Wrapper ultérieur. Inclut les erreurs de demande telles que l'URI non valide, le délai d'expiration inaccessible ou de requête pour l'URI, et les exceptions de sécurité ou autres liées à la demande d'URI VAST.

  • URI Wrapper indisponible - 4xx, 5xx
  • Wrapper n'a pas répondu dans le délai d'attente défini
302 ErrorCodeWrapperLimit

Limite de wrapper atteinte, telle que définie par le lecteur vidéo. Trop de réponses Wrapper ont été reçues sans réponse InLine.

  • Dépassement de la limite de suivi du wrapper
303 ErrorCodeWrapperNoAds

Aucune annonce VAST réponse après un ou plusieurs Wrappers. Inclut également le nombre de réponses VAST vides de secours.

  • Aucune réponse d'annonce linéaire (progressive) reçue par le biais de demandes de wrapper
  • La réponse du wrapper était vide
406 ErrorCodeMissingMezz

La mezzanine était requise, mais elle n'a pas été fournie. Publicité non servie.

  • Implémentations strictes de VAST 4.0 et nécessitant le retour des annonces Mezz. Pourrait être viable pour les dispositifs OTT afin d'assurer la qualité.
  • VAST 4 spécifique
407 ErrorCodeProcessingMezz

Mezzanine a été téléchargée pour la première fois. Publicité non servie.

  • Chaque fois qu'une création valide a été reçue, mais supprimée en raison du traitement de cet événement doit être déclenchée.
  • VAST 4 spécifique
900 ErrorCodeUndefined

Erreur indéfinie

  • Erreur généralisée s'est produite et mal définie par d'autres erreurs
